# Appify.io

Appify.io is a modern, responsive web application designed to showcase, explore, and install mobile applications seamlessly. Built with React and Tailwind CSS, it provides a smooth and dynamic user experience with features such as live search, detailed app information, installation tracking using LocalStorage, and visual review analytics using Recharts.  
Users can browse top apps, filter results instantly, and manage their installed apps â€” all in a clean, intuitive interface. The platform also supports sorting, custom error handling, and interactive toast notifications to enhance user engagement.

## Features
- **Fully Responsive Layout** optimized for all devices.  
- **Dynamic Routing** using React Router DOM for smooth navigation.  
- **Live Search & Sorting** to filter apps in real-time.  
- **App Installation Management** via LocalStorage (Install / Uninstall).  
- **Review Analytics** displayed with Recharts visualizations.  
- **Toast Notifications** for interactive user feedback.  
- **Custom Error Page** and global loading animations for better UX.
